deduce_bound(self, summand_relation, assumptions=None)[source]¶

Given a universally quantified ordering relation over all summand instances, return a bounding relation for the summation. For example, using the summand relation

orall_{k in {m…n}} (a(k) <= b(k)),
we can prove

sum_{l=m}^{n} a(l) <= sum_{l=m}^{n} b(l)

or using the summand relation

orall_{k in {m…n}} (a(k) < b(k)),
and assuming m <= n, we can prove

sum_{l=m}^{n} a(l) < sum_{l=m}^{n} b(l)

split(self, split_index, side='after', assumptions=frozenset())[source]¶

Splits summation over one Interval {a … c} into two summations. If side == ‘after’, it splits into a summation over {a … split_index} plus a summation over {split_index+1 … c}. If side == ‘before’, it splits into a summation over {a … split_index-1} plus a summation over {split_index … c}. As special cases, split_index==a with side == ‘after’ splits off the first single term. Also, split_index==c with side == ‘before’ splits off the last single term. r
